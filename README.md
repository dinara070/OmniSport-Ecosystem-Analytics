**OmniSport Pro** — це комплексна вебплатформа, яка об'єднує спортивний менеджмент, глибоку аналітику даних, генерацію тактичних рішень та інтерактивну фізичну симуляцію біомеханіки спортсменів.

Проєкт створений для демонстрації наскрізної роботи з даними: від введення інформації до її візуалізації на теплових картах та розрахунку кінематичних рівнянь у реальному часі.

---

## ✨ Ключовий функціонал

* 📊 **Smart CRM & Dashboard:** Повноцінна база даних гравців з автоматичним розрахунком рейтингу ефективності (PER - Player Efficiency Rating) на основі алгоритму зважених коефіцієнтів.
* ⚔️ **H2H Scouting (Радарні діаграми):** Пряме візуальне порівняння двох спортсменів за допомогою `matplotlib polar charts`. Дозволяє миттєво оцінити сильні та слабкі сторони.
* 🗺️ **Тактика та Теплові карти (Heatmaps):** Динамічна генерація теплових карт активності на віртуальному полі залежно від фізичних характеристик та виду спорту гравця.
* 🤖 **AI-Тренер:** Автоматична генерація персоналізованого тижневого плану тренувань для усунення слабких місць спортсмена (витривалість, швидкість, сила).
* ⚛️ **Лабораторія Біомеханіки (Data Binding):** Інтерактивний симулятор фізики. Розраховує траєкторію польоту м'яча за законами кінематики. Параметри удару (початкова швидкість, висота) автоматично підлаштовуються під обраного гравця.
* 🎲 **AI-Симулятор Матчів:** Модуль прогнозування результатів зустрічі двох гравців/команд на основі їхнього рейтингу та фактору випадковості.
* 💾 **I/O Data Sync:** Експорт та імпорт бази даних у форматах CSV та JSON.

## 🛠️ Технологічний стек

* **Мова програмування:** Python 3.x
* **Фреймворк:** Streamlit (Fullstack Web UI)
* **Математика та алгоритми:** NumPy (векторизація, розрахунки траєкторій)
* **Робота з даними:** Pandas (DataFrame, агрегація, експорт)
* **Візуалізація:** Matplotlib (Hexbin Heatmaps, Polar Radars, 2D Plots)
